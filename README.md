# fpp

### firefox privacy profile

**fpp is a very lightweight firefox profile**, focused on **portability** and **privacy**.
it may also be used as a launcher for any other firefox profile,
it currently supports: firefox, firefox-esr & librewolf,
and it is also compatible with their flatpak versions.

### USAGE: 

**sh firefox.sh [OPTIONS]**

**note: if no option was supplied, it will just launch fpp**

### OPTIONS

- **-m** to start the most recently used profile
- **-p** to specify a path to a profile
- **-k** to kill the firefox process
- **any url** (one or multiple)
- **RESET** to get a fresh fpp

### TWEAKS

- https-only mode everywhere
- ai enhancements disabled
- all telemetry disabled
- quad9 dns over https

### ADDONS

- UBlock Origin
- Port Authority 
- Disable WebRTC
- Return YouTube Dislike
